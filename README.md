## 下载

### Docker

```
docker pull apachecn0/python-auto-wiki
docker run -tid -p <port>:80 apachecn0/python-auto-wiki
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install python-auto-wiki
python-auto-wiki <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g python-auto-wiki
python-auto-wiki <port>
# 访问 http://localhost:{port} 查看文档
```
