## 下载

### Docker

```
docker pull apachecn0/flygon-auto-wiki
docker run -tid -p <port>:80 apachecn0/flygon-auto-wiki
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g flygon-auto-wiki
flygon-auto-wiki <port>
# 访问 http://localhost:{port} 查看文档
```
